# Mapping Meaning in Lyrics using Embeddings
This repository contains code for mapping meaning in song lyrics using embeddings. The project utilizes natural language processing techniques to analyze and interpret the meaning behind song lyrics by leveraging word embeddings.

Using https://www.kaggle.com/datasets/bwandowando/daily-spotify-top-50-of-60-countries?resource=download




## Create Enviornment (Only Once)
`mamba create -n MMIL python=3.10`

## Open Enviornment
`eval "$(mamba shell hook --shell zsh)" && mamba activate MMIL`
